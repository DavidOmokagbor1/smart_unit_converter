üîß Expo Troubleshooting Steps

Step 1: Clean Everything

# Kill all Expo processes pkill -f expo # Clear Expo cache npx expo r -c # Clear npm cache npm cache clean --force

Step 2: Fix Dependencies

cd SmartUnitConverterExpo rm -rf node_modules package-lock.json npm install npx expo install --fix

Step 3: Try Different Start Methods

# Method 1: Basic start npx expo start # Method 2: Local network npx expo start --lan # Method 3: Localhost only npx expo start --localhost # Method 4: Web version npx expo start --web

Step 4: Check Configuration

Make sure app.json has:

{ "expo": { "newArchEnabled": false, "name": "SmartUnitConverterExpo", "slug": "SmartUnitConverterExpo" } }
