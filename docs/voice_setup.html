Follow these steps to get voice recognition working on your mobile app:

Step 1: Use Chrome Browser

Voice recognition works best in Chrome browser on mobile devices.

  • Download Chrome from App Store (iPhone) or Google Play (Android)
  • Open the app in Chrome browser

Step 2: Allow Microphone Access

When you first try voice input, Chrome will ask for microphone permission:

  • Tap "Allow" when prompted
  • Look for microphone icon in address bar
  • Make sure it's not blocked (crossed out)

Step 3: Test Voice Input

Try the voice input feature:

  • Hold down the "Hold to Record" button
  • Say something like "5 meters" or "10 pounds"
  • Release the button when done speaking
